一括テキスト取り込み

別アプリ（AI等）で写真をテキスト化し、確認のうえ下の書式で貼り付けると、複数問題をまとめて登録できます。図・グラフが本体の問題は「画像: あり」を付け、取り込み後に一覧から写真を後付けできます。

📑 書式の説明とサンプルを見る
・1問ごとに === の行で区切る
・行頭ラベル：分野: / タグ: / 問題: / 選択肢: / 正解: / 解説: / 画像: / 解説画像:
・問題・解説は複数行OK（次のラベルか === までを本文として扱う）
・選択肢は 1. / 1) / ① など可。数は自由（2つ以上推奨）
・正解は番号（例 3、複数正解は 2,4）。空でも取り込み可（後で一覧から設定）
・タグ・解説は任意。コロンは全角／半角どちらでもOK、前後の空白も寛容
・図が問題本体の場合は「画像: あり」、解説が画像なら「解説画像: あり」→取り込み後に一覧から写真を添付

問題データ（GitHub配信）

このアプリは、初回起動時にGitHub上の問題データ（questions.json）を自動で読み込みます。問題を追加・更新したら、下のボタンで最新を取り込めます（履歴は保持・重複は除外）。

データの書き出し / 読み込み

全データ（問題・履歴・画像）を1つのJSONにまとめます。iPhoneとPCで手動同期できます。読み込みは「マージ（合体）」方式なので、既存の履歴は消えません。
